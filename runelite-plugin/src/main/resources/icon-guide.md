# OSRS Guru Plugin Icon

## Requirements
- 16x16 pixels PNG
- Transparent background recommended
- Gold/brown color scheme matching OSRS Guru branding

## Quick Generate (using any tool)
1. Open Canva / Photoshop / GIMP
2. Create 16x16 canvas
3. Draw a simple gold circle (#d4af37) with "G" text
4. Export as PNG → save as `osrsguru-icon.png`

## Alternative: Use OSRS Guru Favicon
Resize the favicon from https://osrsguru.com/favicon.ico to 16x16

## Code Fallback
The plugin has a built-in fallback icon if the resource file is missing (see OSRSGuruPlugin.java:createDefaultIcon()). So you can build and test without an icon first.
