① 赚钱前必知：OSRS 经济系统基础

GP 是 OSRS 的一切

在《Old School RuneScape》中，GP（Gold Pieces） 是核心货币。你用它买装备、买材料、买技能升级、甚至买会员（通过 Bond）。理解经济系统是赚钱的第一步。

核心概念

概念中文解释重要性
Grand Exchange (GE)中央交易市场，位于 Varrock，所有玩家在此买卖物品⭐⭐⭐⭐⭐
GE Tax交易税，卖家需缴纳 1% 手续费（上限 5M GP）⭐⭐⭐
Bond可交易的会员凭证，当前约 8.5M GP/个，兑换 14 天会员⭐⭐⭐⭐⭐
OSRS Wiki GE 价格实时价格参考网站，赚钱前先查价⭐⭐⭐⭐
GP/小时 (GP/Hr)衡量赚钱效率的核心指标⭐⭐⭐⭐⭐
Supply & Demand供需关系决定物品价格，热门 Boss 掉落物价格波动大⭐⭐⭐⭐

赚钱三大原则

  1. 时间就是金钱：同一个小时，做不同的事收益差距可达 100 倍。把时间花在高效方法上。
  2. 本金很重要：100K GP 的倒卖利润远不如 10M GP 的倒卖利润。先积累本金。
  3. 多样化收入：不要只靠一种方法。每天做 Herb Run + 打 Boss + 倒卖，总收入更稳定。
💡 提示：赚钱前先去 OSRS Wiki 价格页面 查一下你想卖的东西的实时价格，避免在 GE 上挂错价格亏钱。

② F2P 新手赚钱法（0–500K GP）

F2P 赚钱限制

F2P（Free to Play）玩家只有约 20% 的地图和一半技能可用，赚钱效率远不如会员。但 F2P 阶段是积累 第一桶金 的关键时期。以下方法按优先级排序。

方法一：收集鸡毛（Feathers）

在 Lumbridge 或 Falador 的鸡场杀鸡，收集 Feathers（羽毛）。虽然单根不值钱，但杀鸡速度快且无等级要求。

项目详情
要求无
GP/小时~15-25K
操作去 Lumbridge 鸡场杀鸡 → 捡 Feathers（每次 5-20 根）→ 到 GE 卖出
优点零门槛，无竞争，可挂机
缺点效率低，只适合刚出新手岛的玩家

方法二：收集贝壳（Big Bones / Cowhides）

杀牛收集 Cowhides（牛皮） 和 Big Bones（大骨头）。牛皮可以用来制作 Leather Armor，是 F2P 最稳定的材料收入来源。

项目详情
要求战斗等级 10+
GP/小时~30-50K
操作去 Lumbridge 或 Falador 的牛场杀牛 → 收集 Cowhide 和 Bones → 到 GE 卖出
推荐地点Lumbridge 牛场（新手友好）、Falador 农场（牛多）
优点同时练战斗技能，牛皮需求量大
缺点竞争较多（新手区人多）

方法三：制作 Iron Bars（铁锭）

收集铁矿石和煤矿石（或直接用铁矿石），在 Furnace 熔炼成 Iron Bars。制作 Iron Bars 是 F2P 锻造入门，收益稳定。

项目详情
要求15+ Mining（挖矿）
GP/小时~40-60K
操作在 Barbarian Village 矿场挖 Iron Ore → 到 Lumbridge 或 Al Kharid Furnace 熔炼 → 卖 Iron Bars / 或做成 Iron Platebody 卖
推荐地点Barbarian Village 矿场（铁矿多且近银行）
优点同时练挖矿和锻造，稳定收入
缺点熔炼速度慢，长时间重复操作

方法四：收集实心绳（Swamp Tar / Worms）与杂项

收集 Lumbridge Swamp 的 Swamp Tar（沼泽焦油）或钓鱼获取 Anchovies 等杂项材料。虽然单次收益不高，但胜在随时可以做。

项目详情
要求无
GP/小时~10-25K
操作在 Lumbridge Swamp 收集 Swamp Tar → 在 GE 卖出
优点极低门槛，适合起步
缺点效率低，只推荐刚起步时过渡用

方法五：GE 倒卖基础（Merchanting / Flipping）

利用 Grand Exchange 的价差低买高卖。即使是 F2P 玩家，也能通过倒卖热门物品赚钱。

项目详情
要求至少 10K+ GP 本金
GP/小时~50-200K（取决于本金和市场经验）
推荐物品Rune Scimitar、Lobster、Iron Ore、Cowhide
操作找出交易量大且价差大的物品 → 低于市价 5% 下单买入 → 高于市价 5% 卖出
优点可远程操作（手机也能做），不需任何技能等级
缺点需要本金，价格波动有风险
💡 新手建议：先用方法一和方法二攒到约 50K GP，然后用这 50K 开始做 GE 倒卖。倒卖赚到的钱再投入购买会员 Bond（约 8.5M GP），进入会员期后赚钱效率翻数倍。

③ F2P 最佳赚钱法 — Obor/Bryophyta 击杀与进阶

F2P 唯二的 Boss

F2P 玩家有两个 Boss 可以击杀：Obor（巨人 Boss）和 Bryophyta（植物 Boss）。掉落装备在 F2P 中非常值钱。

Obor 攻略

Obor 位于 Edgeville Dungeon，需要 Giant Key 才能进入。Giant Key 通过击杀 Hill Giants（山丘巨人）随机掉落。

项目详情
要求40+ Combat Level、Rune 装备
GP/小时~100-200K
关键掉落Rune Full Helm、Rune Chainbody、Rune Kiteshield、Hill Giant Club（稀有，价值约 500K）
操作进入 Edgeville Dungeon → 击杀 Hill Giants 直到掉 Giant Key → 进入 Obor 房间 → 击杀 Obor → 捡掉落
推荐策略在 Edgeville Dungeon 杀 Hill Giants 收集钥匙，攒 5-10 把钥匙后一次性打完

Bryophyta 攻略

Bryophyta 位于 Varrock Sewers，需要 Bryophyta's Essence 召唤 Boss。通过杀 Moss Giants 掉落。

项目详情
要求40+ Combat Level、Rune 装备、抗毒药水
GP/小时~150-250K
关键掉落Rune Scimitar、Rune 2h Sword、Bryophyta's Staff（稀有，价值约 800K）
操作在 Varrock Sewers 杀 Moss Giants 掉 Essence → 在指定位置召唤 Bryophyta → 击杀 → 捡掉落
推荐策略带 Antipoison 药水，Bryophyta 会发射毒弹

F2P 进阶：铁匠铺任务（Smithing Routes）

通过 F2P 的 Blast Furnace（高炉）虽然只有会员能用，但 F2P 可以通过 Superheat Item 魔法 提升熔炼效率。

项目详情
要求43+ Magic（Superheat Item 魔法），挖矿 15+
GP/小时~60-80K
操作挖 Iron Ore → 使用 Superheat Item 魔法同时熔炼 Iron Bars → 到 GE 出售
优点同时练魔法和锻造，比普通熔炼快 2 倍
缺点需要大量 Nature Runes（成本较高）

F2P 高级物品倒卖

F2P 玩家的 GE 交易限额较低，但某些高价值物品利润可观：

物品买入价卖出价每单利润

Rune Platebody~37K~39K~1.5-2K

💡 Bond 目标：在 F2P 阶段通过以上方法攒到 8.5M GP 买第一个 Bond！如果每天玩 3-4 小时，用 Obor/Bryophyta + 倒卖的组合，大约需要 2-4 周。

④ 会员初级赚钱 — Herb Run / Birdhouse / Slayer 掉落

成为会员后的第一个赚钱动作

买了会员之后，你立即解锁了 F2P 玩家无法接触的赚钱世界。以下三个方法是会员前期 回报率最高 的赚钱方式。

方法一：Herb Run（药草种植）— 被动收入之王

Herb Run 是 OSRS 中 性价比最高的被动收入。你只需花 5 分钟跑几个农场种药草，然后等 80 分钟回来收成。

项目详情
要求32+ Farming、完成 Fairytale II 部分任务（解锁 Fairy Rings）
推荐药草Ranarr（赚最多）、Toadflax（稳定）、Snapdragon（高收益高风险）
GP/趟~40-80K（取决于药草种类和运气）
GP/天（6 趟）~240-480K
每趟耗时~5-8 分钟

最佳 Herb Run 路线：

  1. Catherby — 使用 Catherby Teleport 或 Camelot Teleport 跑到农场
  2. Falador — 使用 Falador Teleport 或 Explorer's Ring 2+ 传送
  3. Ardougne — 使用 Ardougne Cape 1+ 传送，或 Ardougne Teleport
  4. Port Phasmatys — 使用 Ectophial 传送
  5. Trollheim — 使用 Trollheim Teleport（需要完成 My Arm's Big Adventure）
  6. Weiss — 使用 Weiss Teleport（需要完成 Making Friends with My Arm）
  7. Farming Guild — 使用 Skills Necklace 传送

建议种 Ranarr（种子 ~25K，收成 6-12 个药草，每个 ~11K，利润约 40-100K/趟）。

方法二：Birdhouse Run（鸟屋）— 被动 Hunter 赚钱

Birdhouse Run 与 Herb Run 搭配，每 50 分钟收一次 Birdhouse，获得 Bird Nests 和各种种子。

项目详情
要求5+ Hunter、完成 The Dig Site Quest（部分）
GP/趟~20-40K
GP/天（10 趟）~200-400K
每趟耗时~3-5 分钟
推荐鸟屋Mahogany Birdhouse（最高级）、Yew Birdhouse（次选）

最佳 Birdhouse 路线：

  1. 准备 Clockwork（每个鸟屋需要 1 个）和 Logs（依鸟屋等级而定）
  2. 使用 Fairy Ring 传送 到各个鸟屋位置
  3. Fossil Island 北岸、南岸、西岸、东岸各有一个 Birdhouse 位置
  4. 放置 Birdhouse → 收成 → 重复

方法三：Slayer 掉落赚钱（中前期）

Slayer 技能不仅是练战斗的好方法，某些怪物掉落的物品价值极高。

怪物Slayer 等级关键掉落GP/小时

Kurasks70Leaf-bladed sword、Rune 装备~300-500K
Wyverns72Wyvern bones、Wyvern visage（稀有 ~5M）~400-700K

💡 最佳组合：每天做 Herb Run + Birdhouse Run 循环，早上起来先跑 2 轮（10 分钟），白天每隔 1-2 小时跑一次，睡前再跑一次。这样每天被动收入约 500K-1M GP，完全免费。

⑤ 会员中级赚钱 — Barrows / Zulrah / Vorkath / Giant Seaweed

进入中期赚钱阶段

当你的战斗等级达到 70+、拥有合理装备后，可以开始挑战中期 Boss 和高价值 Skilling 项目。这些方法每小时收益可达 1-3M GP。

Barrows — 入门级 Boss 赚钱

Barrows 是新手最容易上手的中期赚钱方式，位于 Morytania。需要完成 Priest in Peril 任务解锁区域。

项目详情
要求50+ Magic、50+ Ranged、Iban's Staff（建议）、完成 Priest in Peril
GP/小时~500K-1.2M
关键掉落Barrows 装备六件套（每件约 100K-2M，如 Karil's / Ahrim's / Dharok's）
操作进入 Barrows 地下 → 击杀 6 个兄弟 → 打开宝藏室获得奖励
推荐装备Iban's Staff（Magic）、Rune Crossbow（Ranged）、Dragon Scimitar（Melee）
每趟平均利润~50-100K（含普通掉落，不含稀有 Barrows 装备）

Zulrah — 技巧型 Boss 赚钱

Zulrah 是位于 Karamja 东南方的蛇形 Boss，需要较高的操作技巧。掉落蛇鳞和稀有武器。

项目详情
要求75+ Ranged、75+ Magic、完成 Regicide 任务
GP/小时~2-4M（平均）、熟练玩家可达 5M+
关键掉落Zulrah Scales（每次必掉约 100-500 个，价值 ~100-200K）、Magic Fang（~5M）、Serpentine Visage（~2M）、Tanzanite Fang（~3M）、Uncut Onyx（~2M）
操作学习 Zulrah 的 4 种 Rotation → 装备正确 Prayers → 切换 Magic/Ranged 攻击 → 循环 50-100 次
推荐装备Trident of the Swamp（Magic）、Blowpipe（Ranged）、Ahrim's/Karil's 装备
学习曲线需要 5-10 小时练习才能稳定击杀

Vorkath — 稳定高收入 Boss

Vorkath 是 Dragon Slayer II 任务解锁的蓝龙 Boss，是中期最稳定的赚钱 Boss 之一。

项目详情
要求75+ Ranged、75+ Magic、完成 Dragon Slayer II
GP/小时~2.5-4M
关键掉落Superior Dragon Bones（每次必掉 ~2-4 个，价值 ~20-40K）、Vorkath's Head（稀有 ~5M）、Dragonbone Necklace（~1.5M）、Assembler
操作熟练掌握 Vorkath 的攻击模式 → 躲避冰冻攻击 → 用 Crumble Undead 杀 Zombie → Woox Walk 技巧（进阶）
推荐装备Dragon Hunter Crossbow（核心）、Salve Amulet (e)、Void Knight 装备

Giant Seaweed 种植 — 被动中型收入

在 Fossil Island 种植 Giant Seaweed 是一种稳定的被动收入，每 40 分钟可收成一次。

项目详情
要求23+ Farming、完成 The Bone Voyage 任务
GP/趟~30-50K
GP/天（8 趟）~240-400K
操作使用 Seaweed Spore 在海边种植 Giant Seaweed → 40 分钟后收成 → 每次获得 6-18 个 Seaweed
搭配建议与 Herb Run 同时进行，最大化 Farming 经验
💡 中期赚钱建议：从中期开始，推荐 Vorkath 作为主攻赚钱方式，因为它的收益稳定且操作相对简单。Barrows 适合刚开始打 Boss 的玩家练手。Zulrah 收益更高但学习曲线陡峭。

⑥ 会员高级赚钱 — Raids 1-3 / GWD / Nightmare

进入高级赚钱阶段

当你的战斗等级达到 100+、装备总价值超过 100M GP 时，可以挑战 OSRS 中收益最高但门槛也最高的内容。

Raids 1：Chambers of Xeric（CoX）

CoX 是位于 Mount Quidamortem 的三人副本，需要多种 80+ 技能。

项目详情
要求80+ 攻击/力量/防御、80+ Ranged、80+ Magic、50+ 所有其他相关技能
GP/小时（三人速刷）~3-6M
关键掉落Twisted Bow（~1.5B）—— 全游戏最贵物品！、Kodai Wand（~100M）、Ancestral Robes（每件 ~50-100M）、Dragon Claws（~80M）
推荐人数3 人（效率最高）
学习曲线需要 10-20 小时团队配合学习
推荐装备Dragon Hunter Lance、Trident of the Swamp、Blowpipe、Void / Justiciar

Raids 2：Theatre of Blood（ToB）

ToB 位于 Morytania，是 OSRS 中难度最高的副本，但也是收益最高的之一。

项目详情
要求90+ 所有战斗技能、高操作水平
GP/小时（4-5 人）~4-8M
关键掉落Scythe of Vitur（~800M）、Rapier（~200M）、Justiciar 套装（每件 ~50-100M）、Avernic Defender Hilt（~200M）
推荐人数4-5 人
学习曲线极陡峭，需要 50+ 小时练习才能稳定通关
推荐装备Scythe of Vitur（核心）、Rapier、Blowpipe、全 BiS 装备

Raids 3：Tombs of Amascut（ToA）

ToA 是 2022 年发布的沙漠副本，2026 年仍是热门赚钱方式。

项目详情
要求80+ 所有战斗技能、70+ 其他技能
GP/小时（300+ Invocation 专家模式）~3-6M
关键掉落Shadow of Tumeken（~1.2B）—— 最贵法杖！、Masori 套装（每件 ~50-200M）、Lightbearer（~30M）、Osmumten's Fang（~30M）
推荐人数2-4 人（solo 也可）
学习曲线中等难度，比 ToB 简单但比 CoX 复杂
推荐装备Fang、Blowpipe、Trident、各 Boss 针对性装备

God Wars Dungeon（GWD）

GWD 位于 Trollheim 的冰雪地下城中，有四个 Boss：General Graardor、Kree'arra、Commander Zilyana、K'ril Tsutsaroth。

项目详情
要求70+ 所有战斗技能、完成 Troll Stronghold 任务
GP/小时（组队 3-4 人）~2-4M
关键掉落Bandos Chestplate（~35M）、Armadyl Crossbow（~50M）、Armadyl Chainskirt（~40M）、Saradomin Sword（~6M）、Zamorak Hilt（~10M）
推荐 BossGraardor（最容易上手）、Zilyana（掉最值钱物品）
推荐人数3-4 人组队，坦克 + DPS + 奶妈

Nightmare / Phosani's Nightmare

Nightmare 位于 Sisterhood Sanctuary，是 2020 年发布的高难度 Boss。Phosani's 是单人强化版。

项目详情
要求90+ 所有战斗技能、完成 Darkness of Hallowvale 任务
GP/小时（Phosani 单人）~3-6M
GP/小时（Nightmare 5 人组队）~2-4M
关键掉落Nightmare Staff（~100M）、Inquisitor 套装（每件 ~50-200M）、Slepey Tablet（~30M）
学习曲线陡峭，Phosani 单人模式需要极高操作
💡 高级赚钱顺序：推荐按这个路线进阶——GWD 组队打装备 → CoX 三人速刷 → ToA 专家模式 → ToB 组团 → Phosani 单人。每一个阶段都需要前面积累的装备和资金支持。

⑦ 每日任务循环 — 30分钟日常赚钱流程

高效的每日赚钱循环

下面的 30 分钟日常流程 可以帮你每天稳定赚取 500K-1.5M GP，不需要长时间在线。每天早晚各做一次即可。

⑧ GE 倒卖全攻略 — 低买高卖赚钱术

什么是 GE 倒卖 (Flipping/Merchanting)

Grand Exchange 倒卖是 OSRS 中 最稳定的赚钱方式之一，不需要任何技能等级，只需要本金和市场判断力。核心逻辑：低价买入 → 高价卖出，赚取差价。

倒卖三大策略

策略一：快速倒卖（High Volume Flipping）

选交易量极大的物品，大量买入卖出，每件赚少量差价但靠数量取胜。

物品日交易量平均差价建议日交易量每日利润（100 次交易）
Cannonballs~5M+~2-4 GP5K-10K 个~10K-40K
Rune Scimitar~200K+~500-1K GP200-500 把~100K-500K
Prayer Potion (4)~150K+~200-500 GP500-1K 瓶~100K-500K
Shark~300K+~50-100 GP2K-5K 条~100K-500K
Dragon Bones~100K+~100-200 GP1K-2K 个~100K-400K

策略二：高利润慢速倒卖（Low Volume / High Margin Flipping）

选择高价值但交易量小的物品，挂低价买入、高价卖出，每次利润高但需要耐心。

物品日交易量差价范围每单利润最佳挂单策略
Bandos Chestplate~500~500K-2M~300K-1.5M低 1% 买 → 高 1% 卖，挂 6-24 小时
Armadyl Crossbow~300~500K-3M~500K-2M低价挂买，观察价格走势
Dragon Hunter Lance~100~1M-5M~1M-3M需要耐心，可能数天才能成交
Toxic Blowpipe~2K~50K-200K~30K-150K最适合中级倒卖，量价平衡

策略三：价格波动倒卖（Merching / Investing）

利用大更新、新内容发布造成的价格波动提前买入，更新后卖出。

  1. 关注游戏新闻：Jagex 发布每周更新预告。新 Boss 或装备更新会带动相关材料价格暴涨。
  2. Bond 价格周期：Bond 价格通常在月初较低（会员续费高峰后）、月末较高（需求增加）。在月初买 Bond 存着，月末卖出。
  3. 季节性波动：暑假和节假日期间在线玩家多，材料需求上升，价格往往较高。

倒卖工具推荐

  • RuneLite GE Price Plugin：实时查看物品价格，方便快速决策
  • OSRS Wiki GE 价格页面：查看 30 天/90 天/180 天价格走势
  • GE Tracker 网站：追踪物品价格历史，分析趋势
  • OSRS Flipping Discord：玩家分享倒卖信息和价格预警

倒卖进阶技巧

  1. 利用 4 个 GE 槽位：每个玩家同时有 8 个买入 + 8 个卖出槽位。合理分配不同物品的挂单。
  2. 计算 GE Tax：卖家需支付 1% 税（最高 5M）。确保差价覆盖税费后仍有利润。
  3. 关注 Margin：差价（Buy Price — Sell Price）至少要达到 2% 以上才值得做。
  4. 不要贪心：设定目标利润后立刻成交，不要等更好的价格。市场随时可能变化。
  5. 量力而行：不要把所有资金都押在一个物品上。分散到 3-5 种物品降低风险。
💡 倒卖新手建议：先从 Cannonballs 或 Rune Scimitar 开始练手，这些物品交易量大、价格透明。用 100K 本金试水，熟悉后再加大投入。绝对不要碰你完全不了解的高价物品。

⑨ 风险管理 — 防骗、安全交易与 RWT 防范

OSRS 诈骗手段全解析

OSRS 中存在大量诈骗手段。以下是 中国玩家最常遇到的骗术 及防范方法。

常见骗术一：双重支付骗局（Doubling / Trust Trade）

骗子声称可以帮你 "翻倍金币"（"Drop trade double" 或 "Doubling GP"）。你给他 1M，他说会返你 2M。结果是 一毛都不会还你。

防范方法：记住一句话——世界上没有免费的双倍金币。任何声称可以帮你翻倍金币的人都是骗子。

常见骗术二：钓鱼链接骗局（Phishing Links）

骗子在游戏中发送看似官方的链接，如 "free.runescape.com-claim.com"。你点进去输入账号密码，账号就被盗了。

防范方法：永远不要点击游戏聊天中的外部链接。Jagex 永远不会在游戏中要求你提供密码。始终通过书签或手动输入网址访问官网。

常见骗术三：GE 偷梁换柱

在面对面交易中，骗子先放正确的物品和金币，在最后的确认瞬间替换成不值钱的物品。利用的是你急于点击确认的心理。

防范方法：确认交易窗口中每一个小字。如果觉得卖家在催促你点击确认，停下来，仔细检查物品和金币数额。

常见骗术四：Luring（拖玩家到危险区域）

骗子诱骗你到 Wilderness 或 PvP 区域，然后同伙击杀你抢装备。常见借口："我带你打一个隐藏 Boss"、"这里有个无限刷金币的 Bug"。

防范方法：永远不要跟陌生人去 Wilderness。任何说带你"打隐藏 Boss"或"刷 Bug"的都是 Luring 骗局。

常见骗术五：假玩家交易（Fake GE Offer）

骗子假装在 GE 以超低价卖稀有物品，诱导你离开 GE 私下交易。实际上是低价物品换你的高价值物品。

防范方法：只在 GE 进行交易。不要接受任何"GE 太贵，我们私下交易"的提议。

安全交易技巧

  1. 使用 GE 进行所有交易：GE 是最安全的交易方式，有系统保护。
  2. 面对面交易时使用"Verify"功能：右键物品选"Verify"可以查看物品详情和市值。
  3. 不要和陌生人交换物品：所谓"我借你装备你借我装备"基本都是骗局。
  4. 设置银行 PIN：在银行设置 PIN 码，增加账号安全性。
  5. 启用 2FA（两步验证）：在 Jagex 账号设置中开启 Authenticator。
  6. 不要使用第三方交易平台：所有非官方的交易平台都有极高的诈骗风险。

RWT（Real World Trading）防范指南

RWT（Real World Trading）是指用真实货币买卖 OSRS 金币或物品。这是 严重违反 Jagex 服务条款 的行为。

为什么不能碰 RWT

  1. 封号风险极高：Jagex 有专门的 RWT 检测系统。一旦发现永久封号，没有任何申诉机会。
  2. 中国玩家风险更大：中文卖金网站是 Jagex 的重点监控对象。很多中国玩家因此被封号。
  3. 被骗风险 100%：卖金网站给你号打币可能被 Jagex 监控到；买金可能付款后对方不发货。
  4. 毁掉游戏体验：用 RWT 来的金币买装备，你会很快失去游戏的乐趣和成就感。

安全合法的替代方案

  • 用游戏内方法赚钱：每个阶段都有适合的赚钱方式（本攻略已覆盖全部）。
  • 用 Bond 支付会员：游戏中赚 GP 买 Bond 而不是用 RMB 买会员。这不仅合法还省钱。
  • 用 PayPal 直接买会员：如果不想花时间赚 Bond 的钱，直接用 PayPal 购买会员（见会员支付指南）。
