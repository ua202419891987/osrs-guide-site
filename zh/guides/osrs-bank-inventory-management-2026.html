OSRS 银行与背包管理指南 2026

1. 什么是银行？为什么你需要它

你的背包（Inventory）只能装28个物品。仅此而已。玩20分钟后，背包就会满。你需要一个地方存放其他所有东西 —— 这就是银行（Bank）的用途。

🏦 银行的作用

  • 存储几乎无限的物品（2,304个槽位 —— 对大多数玩家来说就是无限的）
  • 可以从游戏中任意银行柜台 / 银行NPC / 银行箱子访问
  • 可以精确取出某个活动所需的物品
  • 可以存入长时间打怪或打Boss的战利品

如何找到银行

每个主要城镇都有银行。离新手岛最近的银行：

  • Lumbridge（鲁姆桥）：Lumbridge Castle二楼有银行箱子（新手最方便的选择！）
  • Varrock（瓦洛克）：西银行和东银行（非常受欢迎）
  • Falador（法拉多）：城镇广场南侧
  • Edgeville（边缘村）：就在Wilderness边界北边（高级内容很方便）
💡 在世界地图上标记你的第一个银行

一到Lumbridge Castle，打开世界地图，找到二楼的银行箱子，右键 → "添加标记"。现在你总能看到一个绿色箭头指引你回家。

2. 如何使用银行 — 分步指南

以下是使用银行的具体步骤：

📥 存入物品（把东西放进银行）

  1. 走到银行柜台 / 银行NPC / 银行箱子前
  2. 右键点击银行职员 → "Bank"（或左键点击如果显示"Bank"）
  3. 银行界面打开（显示你存储物品的大窗口）
  4. 存入：点击背包里的物品 → 物品进入银行。或点击"存入背包物品"（右下角）一次性存入所有物品。
  5. 完成后关闭银行（按Esc键或点击X）

📤 取出物品（把东西从银行拿出来）

  1. 打开银行（同上）
  2. 在银行界面中找到你想要的物品
  3. 左键：取出1个
  4. 右键：显示选项：取出1个、取出5个、取出10个、取出全部、取出所有但保留1个
  5. 点击你需要的数量 → 物品出现在背包里
  6. 关闭银行

取出模式：票据 vs 实物

这是银行界面中让很多新手困惑的一个设置：

模式效果何时使用
实物（默认）取出实际物品（每个占1个背包格）取出装备、食物、药水
票据（Note）取出物品为"银行票据"（可堆叠，1格可装最多2,147,483,647个物品）大量出售到Grand Exchange (GE)
⚠️ 新手常见错误：不小心用票据模式取出了物品

你点击"取出全部"拿装备，结果你的盔甲变成了"票据"，无法穿戴。解决方法：右键点击背包里的票据 → "以实物形式取出"。取物品前一定要检查银行界面底部的"票据/实物"切换按钮！

3. 银行标签页（Bank Tabs） — 整理你的游戏人生

银行有8个标签页（后续解锁"银行整理器"功能后可获得更多）。你可以重命名标签页并将物品分配到其中。这对保持井井有条至关重要。

如何设置银行标签页

  1. 打开银行
  2. 点击银行界面上方的标签页名称（如"Tab 1"）
  3. 输入新名称（如"战斗装备"）
  4. 将物品拖入该标签页进行分配
  5. 对全部8个标签页重复此操作

新手推荐标签页设置（F2P）

标签 #名称存放内容
1⚔️ 战斗装备你最好的盔甲套装、武器、盾牌、项链
2🍳 食物熟鱼、面包、各种食物
3💰 贵重物品Rune物品、宝石、昂贵的掉落物
4🎣 技能工具钓鱼竿、鱼饵、斧头、镐
5📦 原材料矿石、原木、生鱼、未烹饪食物
6📜 掉落物/战利品从怪物那里获得的未整理掉落物（稍后整理）
7🎒 备用装备低等级护甲（作为备份保留）
8🔮 任务物品以后可能用到的任务相关物品

4. 背包管理技巧

你的背包只有28个格子。好的背包管理意味着更少跑回银行的次数。以下是最大化背包的方法：

可堆叠物品 vs 不可堆叠物品

类型例子占用格子
可堆叠金币、符文（Runes）、箭矢、弩箭、羽毛1格（最多可堆21亿个同种物品）
不可堆叠食物、武器、盔甲、原材料每个物品1格

📦 背包管理规则（牢记这些）

  1. 先装满背包再去银行。不要拿着10个空格子跑回银行。先多杀10个怪物或多采集10个资源。
  2. 丢弃低价值物品。如果物品价值低于100 GP且背包已满，直接丢弃。你的时间比卖掉它赚的GP更有价值。
  3. 使用"Shift+点击丢弃"设置。在 设置 → 游戏玩法 → 控制 中启用。让丢弃物品速度快10倍。
  4. 随身携带传送符文。在背包里保留3-5个传送符文（Law + Air + Earth），方便快速返回银行。

5. 哪些物品该保留 vs 丢弃/出售

作为新手，你会经常问自己："这个东西值得留还是可以丢掉？" 以下是一个决策框架：

物品价值（GE价格）操作例子
<100 GP丢弃（不值得占背包空间）青铜（Bronze）物品、骨头（除非练Prayer）、低级草药
100-1,000 GP出售到GE，或背包满时丢弃铁质物品、牛皮、低级鱼
1,000-10,000 GP始终保留（存入银行或出售到GE）钢质/秘银（Mithril）物品、未切割宝石、草药
>10,000 GP一定要存入银行或出售（这些积累很快！）Rune物品、线索卷轴、龙装备、草药（如Grimy Ranarr等）
💰 "牛皮技巧" — 你的第一个收入来源

在Lumbridge附近杀牛。收集牛皮（每个价值约100-200 GP）。装满背包（28张牛皮）。存入银行。攒到50-100张后，到Grand Exchange出售。这是经典的F2P新手赚钱方法。简单、无属性要求，还能边打边练Combat！

6. 银行空间 — 当你的位置用完时

银行有2,304个物品槽位（8个标签页 × 每标签288个物品）。大多数玩家永远不会装满，但如果你是个囤积狂，以下是解决办法：

如何释放银行空间

  • 将物品转换为银行票据：以票据形式取出物品（它们在银行里如果被标记为票据也能堆叠）。等等 —— 实际上，物品在银行里已经可以堆叠了。票据仅用于取出。
  • 丢弃/出售低价值物品：诚实地问自己 —— 你真的会去用那500支青铜箭吗？不会？那就丢掉。
  • 使用"占位符"功能：在银行设置中，启用"占位符（Placeholders）" —— 这会保留一个显示物品名称的空槽位，让你知道你拥有它但不占用实际空间。
  • 获取会员"银行整理器"：会员可以解锁额外的银行标签页（通过某些解锁最多可获得8+个）。更多标签页 = 更多整理选项。

7. 你应该知道的实用银行功能

搜索栏（银行界面顶部）

输入物品名称 → 银行只显示匹配的物品。不用再滚动500个物品找你的Rune scimitar。

存入背包 / 存入穿戴物品按钮

在银行界面底部：

  • "存入背包物品"：存入背包中的所有物品。一键完成。
  • "存入穿戴物品"：脱下所有装备并存入银行。战斗结束后非常方便。

银行PIN码（安全功能）

设置银行PIN码，这样即使有人黑进你的账号，也不能立刻偷走你的银行物品。设置路径：设置 → 显示与声音 → 银行PIN码。

⚠️ 不要忘记你的银行PIN码

如果你设置了银行PIN码但忘记了，你需要走Jagex账号恢复流程。非常麻烦。把它写在安全的地方（但不要写在电脑上的文件里）。

8. 新手银行标签页设置（分步操作）

让我们正确设置你的银行标签页。按以下步骤操作：

  1. 前往任意银行并打开它
  2. 点击"Tab 1" → 重命名为"⚔️ 装备"
  3. 点击"Tab 2" → 重命名为"🍳 食物"
  4. 点击"Tab 3" → 重命名为"💰 出售/GP"
  5. 点击"Tab 4" → 重命名为"🎣 工具"
  6. 点击"Tab 5" → 重命名为"📦 原材料"
  7. 点击"Tab 6" → 重命名为"🔮 任务物品"
  8. 点击"Tab 7" → 重命名为"📚 备用装备"
  9. 点击"Tab 8" → 重命名为"❓ 未整理"
  10. 现在将银行中的每个物品拖入对应的标签页
  11. 完成！你的银行现在已整理好了。

9. 常见的银行使用错误

⚠️ 银行使用5大错误
  1. 不使用银行标签页：你有100多个物品全堆在Tab 1里毫无组织。你花了5分钟找传送符文。赶紧设置标签页！
  2. 不小心以票据形式取出物品：你取出装备变成了票据，无法穿戴。检查票据/实物切换按钮！
  3. 不够频繁地去银行：你背包满了还继续玩，然后死了东西全丢。训练时每15-20分钟去一次银行。
  4. 囤积无价值物品：你存着2024年的500支青铜箭。每支只值5 GP。直接丢掉。
  5. 忘记设置银行PIN码：有人黑进你的账号，30秒内清空了你的银行。别让这种事发生在你身上 —— 今天就设置PIN码。

10. 接下来做什么

你现在已经理解了银行和背包管理！以下是接下来要做的事：

✅ 你的银行行动计划

  1. 马上去Lumbridge Castle的银行箱子
  2. 按上面的命名方案设置8个银行标签页
  3. 将你当前的物品整理到正确的标签页
  4. 在设置中启用"Shift+点击丢弃"（让背包管理速度快10倍）
  5. 设置银行PIN码来保护安全
