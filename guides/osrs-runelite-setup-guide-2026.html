1. Why Every OSRS Player Uses RuneLite

In 2026, over 90% of active OSRS players use RuneLite as their primary game client. The official Jagex client is functional, but RuneLite transforms the game with features that feel like they should be built-in.

🎯 What RuneLite Gives You That the Official Client Doesn't

  • Quest Helper: Shows you exactly where to go, what to click, and what items to bring — eliminates the need for external quest guides
  • GPU Rendering: Smooth 60+ FPS in crowded areas where the official client drops to 15-20 FPS
  • XP Trackers: Real-time XP/hour, estimated time to next level, and session tracking
  • Ground Items: Highlights valuable drops, hides junk loot, shows item values
  • Menu Entry Swapper: Left-click to use/equip instead of right-click options
  • 117 HD Mode: Beautiful lighting, shadows, and texture filtering (optional plugin)

⚠️ Important: RuneLite is completely legal and approved by Jagex. The official rules only forbid clients that automate gameplay (auto-clickers, macro tools). RuneLite is a visual/UI enhancement client only — completely safe to use on your main account.

2. How to Download & Install RuneLite Safely

There is exactly ONE safe source to download RuneLite. Every other site is either a copycat or malware. Never search "RuneLite download" and click the first result — scammers run fake ad sites.

① Step 1: Go to the Official Website

Open your browser and go to https://runelite.net. Look for the green "Download" button on the homepage. The URL must be exactly runelite.net — not "runelite-download.com", "runelitefree.net", or any other variant.

② Step 2: Run the Installer

Windows: Download RuneLiteSetup.exe and run it. The installer is only 2-3MB — it downloads the rest on first launch. Accept the default installation location. Mac: Download the .dmg file and drag to Applications. Linux: Available via Flatpak or direct download.

③ Step 3: First Launch & Login

After installation, launch RuneLite. It opens the OSRS login screen — log in with your normal OSRS credentials. On first launch, RuneLite downloads additional resources (~100MB). This is normal. Wait for the "RuneLite loaded" message in the top-right corner.

🔒 Security Check: RuneLite is open-source (you can verify the code on GitHub). If you're asked for your bank PIN, authenticator code, or email password — that's NOT RuneLite. Close it immediately. RuneLite only needs your game login, nothing else.

4. Plugin Hub: Hidden Gems You Didn't Know About

The Plugin Hub contains community-created plugins that aren't in the default list. Open RuneLite → Configuration (wrench icon) → Plugin Hub tab. These are less tested but incredibly useful.

① 117 HD (Best Visual Overhaul)

The most popular Plugin Hub plugin. 117 HD adds GPU-accelerated lighting, shadows, water reflections, and viewport bobbing. It makes OSRS look like a modern game while keeping the classic feel. Requires the GPU plugin to be enabled first, and a graphics card with OpenGL 3.3+ support.

② Wilderness Line

Shows the Wilderness ditch, level lines, and the exact tiles where you can teleport. Essential for anyone entering the Wilderness for clues, bosses, or slayer. Saves you from accidentally being 1 level too deep to teleport.

③ Visual Sounds

Adds visual indicators for sound-based events. Shows a screen indicator when you receive a slayer XP drop, hear a boss attack sound, or trigger a clue step. Useful for players who play without sound or have hearing impairments.

④ Farming Tracker

Tracks all your farming patches: what's planted, when it's ready, and estimated yield. Syncs across sessions. Better than the built-in Farming plugin because it supports all patch types and calculates seed-to-herb conversion rates.

5. Best Settings for Performance & FPS

Even on a low-end laptop, RuneLite can run smoothly. Here's the optimal configuration:

① GPU Plugin Configuration

Open RuneLite → Configuration → GPU Plugin. Set Draw Distance to "Medium" (or High if you have a dedicated GPU). Enable Fog Depth at 25-40 to improve performance. Keep Bilinear Filtering enabled for smoother textures. Disable Anti-aliasing if FPS drops below 30.

② Low-End Laptop Optimization

If your computer struggles: Disable the GPU plugin entirely (use CPU rendering). Reduce Draw Distance to "Low" or minimum. Disable Entity Hider for players and pets. Turn off animated sprites (water, fire, etc.). Set game detail to "Low" in OSRS settings. Your FPS should stabilize at 25-35 on most laptops.

③ High-End Desktop Optimization

For a gaming PC: Enable GPU plugin with "Ultra" Draw Distance. Enable 117 HD Mode at 1080p+ resolution. Turn on Anti-aliasing x4. Enable animated textures. Set OSRS graphics to "High" detail. You'll get smooth 60 FPS in almost all areas except the Grand Exchange (which is CPU-limited).

6. RuneLite 117 HD Mode Setup (Optional)

117 HD transforms OSRS with modern graphics. Here's how to install it and whether your computer can run it.

① System Requirements

You need: Windows/Mac/Linux 64-bit. Graphics card with OpenGL 3.3+ support (any GPU from the last 10 years works). 4GB+ RAM. 500MB free disk space. Integrated Intel HD Graphics from 2015 or newer can run 117 HD at "Low" settings at 30 FPS.

② Installation Steps

Open RuneLite → Configuration (wrench icon) → Plugin Hub → Search "117 HD" → Install. Restart RuneLite. After restart, click the new "HD" button on the right side of the game screen. Configure: Preset: "Low" for integrated GPU, "Medium" for entry-level discrete GPU, "High" for gaming PCs. Toggle "Anisotropic Filtering" to x4 for sharper textures.

③ Performance Tips

If 117 HD causes lag: Reduce the "Render Distance" slider. Disable "Smooth Lighting." Set "Texture Detail" to "Low." Turn off "Water Reflections." Enable the GPU plugin with "Compute Shaders" ticked. These settings maintain 90% of the visual improvement while doubling FPS.

7. Common RuneLite Problems & Fixes

🛑 RuneLite won't launch / "Java not found"

Download and install Java 17 from https://adoptium.net. RuneLite 2.x requires Java 17+. Restart your computer after installing Java, then launch RuneLite again. If the problem persists, reinstall RuneLite from runelite.net.

🛑 Black screen on login

This is usually a GPU plugin conflict. Hold Shift while launching RuneLite to boot in Safe Mode. Once loaded, go to Configuration → GPU Plugin and disable it. Restart RuneLite normally. If the screen is still black, disable the "Stretched Mode" plugin.

🛑 RuneLite is laggy / low FPS

Disable the GPU plugin and use CPU rendering. Reduce Draw Distance. Disable Entity Hider for players. Turn off "Animated Textures" in the GPU plugin settings. If using 117 HD, switch the preset to "Low." Lower your OSRS game window resolution.

🛑 "RuneLite is out of date" error

RuneLite auto-updates on launch. If you see this error, close RuneLite completely (check system tray), download the latest installer from runelite.net, and reinstall. The reinstall preserves all your plugin settings. This error is common after OSRS game updates.

🛑 Plugin Hub is empty or won't load

Open Configuration → Plugin Hub tab. If it's blank, click the "Refresh" button in the top-right (circular arrow). If it still fails, restart RuneLite. The Plugin Hub is community-hosted — sometimes the server is temporarily offline. Check back in an hour.

8. Frequently Asked Questions

Q: Is RuneLite allowed by Jagex in 2026?

Yes, RuneLite is fully Jagex-approved. Jagex's rules prohibit bots and automation — not UI enhancements. RuneLite is 100% safe to use on your main account. The developers work with Jagex to ensure compliance.

Q: Can I get banned for using RuneLite plugins?

No — every plugin in the official Plugin Hub is reviewed and approved. Avoid downloading plugins from random websites or Discord servers. Stick to the official Plugin Hub and built-in plugins.

Q: Which RuneLite plugins are banned or risky?

Plugins that automate actions (auto-switch prayers, auto-click, auto-switch gear) are against the rules. These are NOT in the Plugin Hub. If you see "anti-bot detection" plugins, avoid them. The official RuneLite client is safe; modifications are not.

Q: RuneLite vs official Jagex client — which is better?

RuneLite wins for almost everything: better FPS (GPU plugin), quest helper, XP tracking, and 117 HD graphics. The only advantage of the official client is zero setup time. Every experienced player uses RuneLite.

Q: How do I reset RuneLite to default settings?

Close RuneLite. Press Windows+R, type %userprofile%\.runelite\ and press Enter. Delete the "settings.properties" file. Restart RuneLite — it will generate fresh settings. Your plugins will still be installed but reset to defaults.

Q: Does RuneLite work on Mac?

Yes. Download the .dmg file from runelite.net. Requires macOS 10.15 (Catalina) or newer. Some Plugin Hub plugins may have issues on Apple Silicon (M1/M2/M3) — if a plugin doesn't load, check the Plugin Hub comments for Mac-specific setup.

Q: Does RuneLite work on Steam Deck or Linux?

Yes. Install via Flatpak: run "flatpak install net.runelite.RuneLite" in terminal. Or use the direct .jar launcher for manual installation. Works well on Steam Deck in Desktop Mode with touch controls configured.

Q: What's the difference between RuneLite and RuneLite++?

RuneLite++ is an unofficial fork with extra plugins. It is not safe to use on your main account — it violates Jagex's third-party client rules. Stick to official RuneLite. The "++" risk is not worth it.

Q: Can I use RuneLite on mobile?

No — RuneLite is a desktop-only client. Mobile players must use the official OSRS Mobile app. However, you can use the RuneLite desktop client on tablets that run desktop Windows (Microsoft Surface, etc.).

Q: How do I update RuneLite to the latest version?

RuneLite auto-updates. If you need to force an update: close RuneLite completely (check system tray), relaunch from the desktop shortcut. The launcher checks for updates on every start. If the auto-update fails, download the latest installer from runelite.net.
