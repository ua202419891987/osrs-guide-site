2. Installing RuneLite on Windows

Windows is the easiest platform to install RuneLite. Here's the step-by-step:

📥 Method 1: Official Installer (Recommended)

  1. Go to https://runelite.net/ (official website)
  2. Click "Download (Windows)" — this downloads RuneLite-Setup.exe
  3. Run the installer (it will install to C:\Program Files\RuneLite\ by default)
  4. Launch RuneLite from the desktop shortcut or Start menu
  5. Log in with your Jagex account email + password (same as Steam)

📥 Method 2: Portable Version (No Install)

  1. On the download page, click "Portable (.exe only)"
  2. Save the RuneLite.exe file to any folder (e.g., Desktop)
  3. Double-click RuneLite.exe to launch — no installation needed!
  4. This is useful if you want to run RuneLite from a USB drive or don't have admin rights
⚠️ Anti-Virus Warning

Some anti-virus programs (Windows Defender, McAfee) may flag RuneLite as "suspicious" because it modifies the game client. This is a false positive — RuneLite is open-source and safe. Add RuneLite to your exclusions list if needed.

3. Installing RuneLite on Mac

Mac installation is also straightforward. Note: macOS 10.15 (Catalina) or later is required (64-bit support).

📥 Step-by-Step (Mac)

  1. Go to https://runelite.net/
  2. Click "Download (Mac)" — downloads RuneLite.dmg
  3. Open the .dmg file and drag RuneLite.app to your Applications folder
  4. Launch RuneLite from Applications (you may need to right-click → "Open" the first time to bypass Gatekeeper)
  5. Log in with your Jagex account credentials

🍎 Mac-Specific Tips

  • M1/M2/M3 Macs: RuneLite runs natively on Apple Silicon — no Rosetta needed. Performance is excellent (144+ FPS on M2 MacBook Air).
  • Retina Scaling: Go to Settings → Stretch → "Retina (HiDPI)" for crisp UI on high-resolution screens.
  • Trackpad Zoom: Enable Settings → Interface → "Trackpad zoom" to zoom with pinch gestures.

4. Installing RuneLite on Linux

Linux installation is where many players get stuck (as seen on Reddit). Here's how to do it properly:

🐧 Method 1: AppImage (Easiest, All Distros)

  1. Go to https://runelite.net/
  2. Click "Download (Linux AppImage)" — saves RuneLite-x.y.z.AppImage
  3. Make it executable: chmod +x RuneLite-*.AppImage
  4. Run it: ./RuneLite-*.AppImage

🐧 Method 2: Flatpak (Fedora, Ubuntu, Mint)

  1. Install Flatpak if not already installed: sudo apt install flatpak (Ubuntu/Debian) or sudo dnf install flatpak (Fedora)
  2. Add Flathub: flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
  3. Install RuneLite: flatpak install flathub net.runelite.RuneLite
  4. Launch: flatpak run net.runelite.RuneLite (or find it in your app menu)

🐧 Method 3: Arch/Manjaro (AUR)

  1. Install from AUR: yay -S runelite or paru -S runelite
  2. Launch: runelite
⚠️ Linux GPU Rendering Issue

If RuneLite crashes on Linux with a GPU error, force software rendering: launch with RUNELITE_RENDERER=SOFTWARE ./RuneLite.AppImage. Or enable Settings → GPU → "Disable GPU acceleration" inside RuneLite.

5. 10 Essential Plugins for Beginners

RuneLite's plugin hub has 100+ free plugins. Here are the 10 most important ones for new players — install them immediately after your first login:

📋 How to Install Plugins

Click the wrench icon (⚙️) in the top-right corner of the RuneLite window → "Plugin Hub" → search for the plugin name → click "Install".

🔟 Must-Have Plugins

  1. Tile Indicators — Shows the attack range of NPCs (red tiles = they can hit you). Essential for safe spotting and PvM.
  2. Ground Markers — Right-click any tile to mark it (color-coded). Use it to mark bank chesk, quest NPCs, or dangerous spots.
  3. Opponent Info — Shows enemy HP, prayer, gear, and special attack % above their head. Critical for PvP and bossing.
  4. Prayer Helper — Highlights which prayer to use against each NPC. No more guessing!
  5. Idle Detector — Notifies you (with a sound) when your character is idle (not attacking/moving). Great for AFK training.
  6. XPTracker (or XP Updater) — Shows XP/hour, time to next level, and total XP gained in a small overlay.
  7. Inventory Tags — Color-code items in your inventory (e.g., red = food, green = potions). Makes inventory management faster.
  8. Menu Entry Swapper — Changes right-click menu order (e.g., make "Attack" the left-click action for NPCs). Saves thousands of clicks.
  9. Shift Click Deposit/Withdraw — Lets you shift-click items in bank to quickly deposit/withdraw pre-set quantities (e.g., 5, 10, All).
  10. Time Visualizer (or Time Rendering) — Shows cooldown timers on your character (prayer drain, poison tick, etc.) as a visual bar.
💡 Plugin Conflict Warning

Some plugins can conflict with each other (e.g., two plugins both trying to modify the right-click menu). If something looks weird, disable plugins one-by-one to find the culprit. RuneLite is very stable, but too many plugins can cause minor UI glitches.

6. Performance Optimization & FPS Boost

RuneLite can run at 144+ FPS (vs 60 FPS cap on Steam). Here's how to maximize performance:

⚡ Enable GPU Acceleration (Most Important!)

  1. Open Settings (wrench icon → Settings)
  2. Go to "GPU" tab
  3. Check "Enable GPU acceleration"
  4. Set "AA Sampling" to "MSAA 8x" (if your GPU can handle it) or "None" for max FPS
  5. Restart RuneLite for changes to take effect

🖥️ Stretch Mode (Fullscreen, No Black Bars)

By default, OSRS runs at a fixed 765×503 resolution with black bars on widescreen monitors. To fix:

  • Go to Settings → Stretch
  • Check "Stretch" — this scales the game to fill your monitor
  • Set "Scaling mode" to "Retina (HiDPI)" on Mac, or "Linear" on Windows
  • Now OSRS fills your entire screen with no black bars!

7. Transferring Settings from Steam

If you've been playing on Steam and want to keep your custom keybinds, chat settings, and display options, you can transfer them to RuneLite:

🔄 Settings Transfer Steps

  1. On Steam: Log in, go to Settings (wrench icon) → "Export Settings" → save the .json file
  2. On RuneLite: Log in, go to Settings (wrench icon) → "Import Settings" → select the .json file
  3. Your keybinds, chat filters, and display settings will now match your Steam setup!

Note: RuneLite has many more settings than Steam. After importing, explore the Settings panel — you'll find options that don't exist in the Steam client.

💡 Keep Both Clients Installed

There's no harm in keeping both Steam and RuneLite installed. Some players keep Steam as a backup (in case RuneLite has an update bug), or use Steam for afk training on a second account. Your account works on both!

8. RuneLite Frequently Asked Questions

❓ Is RuneLite safe? Can I get banned for using it?

No, you cannot get banned. RuneLite is officially approved by Jagex — it's open-source, doesn't automate gameplay, and is linked from the official OSRS website. Jagex actually employs RuneLite contributors!

❓ Does RuneLite work with my Jagex account?

Yes! RuneLite uses the same Jagex account login as Steam, Mobile, and the official client. Your progress is tied to your account, not the client.

❓ Why is RuneLite capped at 60 FPS for me?

You probably didn't enable GPU acceleration. Go to Settings → GPU → check "Enable GPU acceleration". After restarting, you should get 144+ FPS (if your monitor supports it).

❓ Can I use RuneLite on multiple computers?

Yes! Install RuneLite on all your computers. Your settings can be exported/imported via .json file, and your account progress is always synced via Jagex's servers.

❓ Does RuneLite work on ChromeOS/Chromebooks?

Not directly (ChromeOS is Linux-based but doesn't easily run .AppImage). However, you can use Linux (Beta) on ChromeOS to run the Linux version, or just use the official Android app (sideloaded) on Chromebook.
