1. What Are Bank Tags? (June 2026 Update)

On June 17, 2026, Jagex released a massive game update that, among other things, brought Bank Tags to the official C++ client and mobile. This feature was previously only available through RuneLite's Bank Tags plugin — now it's built directly into the game.

🏷️ What Bank Tags Do

  • Create up to 20 custom tags that sit alongside your main bank
  • Each tag acts as a smart filter — it only shows the items you've assigned to it
  • One item can belong to multiple tags at once (e.g., Dragon Warhammer can be in both "PvM" and "Raids")
  • Moving items inside a tag does NOT affect your main bank layout
  • Withdrawing from a tag does reduce the item count in your main bank — tags are views, not separate storage
📱 Available On

Bank Tags work on all three platforms: the Official C++ Desktop Client, the Mobile App, and RuneLite (RuneLite's version has been around longer but works alongside the new system). Make sure your client is updated to the latest version to access the feature.

Why You Should Use Bank Tags

Before Bank Tags, players had to scroll through hundreds of items in 8 bank tabs trying to find what they needed. With tags, you can create activity-specific views. Want to gear up for Vorkath? Click your "Vorkath" tag and see exactly the 15 items you need — no distractions, no scrolling.

Source: Official News Post — June 17, 2026

2. How to Create & Edit Bank Tags — Step by Step

① Creating Your First Tag

  1. Open any bank in the game
  2. Look at the left sidebar of the bank interface — you'll see your main bank tab icons
  3. Click the (+) button at the top-left of the bank interface
  4. Type a name for your tag in the chatbox (up to 25 alphanumeric characters)
  5. A search box opens — type the name of an item to use as your tag icon (almost every tradeable and untradeable item works)
  6. Confirm — your new tag is created!

② Adding Items to a Tag

  1. With your tag selected, open your main bank view
  2. Drag items from the main bank onto the tag icon in the sidebar
  3. The item now appears in that tag's view
  4. Repeat for all items you want in this tag

③ Organizing Items Inside a Tag

  1. Click your tag to enter its view
  2. Drag items freely to any position — you can leave empty spaces between items
  3. Create entire blank rows if you want visual separation between gear, food, and potions
  4. Your layout is saved automatically

④ Duplicating, Renaming & Deleting Tags

  1. Duplicate a tag: Hold Shift and drag a tag icon — this copies the entire tag layout (useful for creating variations)
  2. Rename a tag: Right-click the tag icon → "Rename"
  3. Remove a single item: Right-click the item in tag view → "Remove from tag"
  4. Delete a tag: Right-click the tag icon → "Delete tag" (you'll get a confirmation prompt)
  5. Reorder tags: Drag tag icons up and down in the sidebar
💡 Pro Tip: Copy Layouts From Community Members

Content creators like Megan (Elite Clue Scrolls), MMORPG (Muspah/Zulrah), Zecookies (Farm Runs), and Muts (Deep Wildy) shared their personal bank tag layouts in the official news post. These are great starting points to copy and customize.

3. How to Share & Import Bank Tag Layouts

The official client does not support one-click import/export of bank tags, but the community has built excellent tools to bridge this gap.

🌐 Community Platforms for Sharing

  • RuneTags.com — Browse and share bank tags, ground markers, and inventory setups. Upload your own layouts or download community favorites.
  • BankLayouts.com — A dedicated tool to create, edit, and share bank tag layouts. Browse a catalog of premade layouts or build your own.
  • Reddit r/2007scape — Search for "bank tag layout" to find screenshot-based layouts that you can replicate manually.
  • Discord communities — Servers like We Do Raids and the OSRS Official Discord have dedicated channels for sharing bank setups.

How to Share Your Layout

To share your layout with others:

  • Screenshot method: Take a screenshot of your tag view and post it — other players replicate it manually
  • Item list method: Write out the items in order (top to bottom, left to right) and share as a text list
  • RuneLite export: If you use RuneLite, the Bank Tag Layouts plugin allows exporting layouts as text/code that others can import
💡 Why Share?

The OSRS community has thousands of optimized layouts for every boss and activity. Instead of reinventing the wheel, find a layout from a top PvMer, copy it, and tweak it to your needs. It saves hours of trial and error.

4. Bank Tag Layout Plugin — Advanced Features (RuneLite)

If you use RuneLite, the Bank Tag Layouts plugin from the Plugin Hub adds powerful features beyond the official client's capabilities.

① Installing the Layout Plugin

  1. Open RuneLite and press the wrench icon to open settings
  2. Go to Plugin Hub
  3. Search for "Bank Tag Layouts" and click Install
  4. Enable the plugin — you'll now see a "Enable Layout" option when right-clicking a tag

② The Presets System — One-Click Gear Setup

  1. Equip all your gear and fill your inventory for a specific activity (e.g., TOA)
  2. Go to Bank Tag Layouts plugin settings → switch Style to "Presets"
  3. Click the Auto-Layout button (white button, bottom-right of bank interface)
  4. The plugin automatically copies your current equipment + inventory into the tag layout, matching the grid positions exactly
  5. Now every time you open this tag, you see exactly what you need, where you expect it

③ Advanced Features

  • Disable Layout: Right-click the primary item in a tag → "Disable Layout" to freely drag items anywhere
  • Custom spacing: Create gaps and blank rows for visual organization
  • Export/Import: Share your layouts as text code that others can import into RuneLite
  • Combine with Placeholders: Use game's built-in placeholders to maintain positions in your main bank too

10. Bank Tags FAQs

How many bank tags can I create?

You can create up to 20 bank tags on the official client. RuneLite users with the Bank Tags plugin can create unlimited tags.

Do bank tags use extra bank space?

No. Bank tags are just filters/views of your existing bank items. They do not add or remove bank slots. Items in tags are still in your main bank.

Can one item be in multiple tags?

Yes. A single Dragon Warhammer can be in your PvM tag, Raids tag, and Slayer tag simultaneously. Tags do not duplicate items — they just create alternative views.

Does organizing a tag affect my main bank?

No. Moving items around inside a tag view does NOT change the order in your main bank. Tags and main bank layouts are independent.

Do bank tags work on mobile?

Yes! Bank Tags were released simultaneously on both the Official Client and Mobile on June 17, 2026. The mobile interface has been optimized for touch-based tag management.

Can I share my bank tag layout with friends?

There is no official one-click share button, but you can share via screenshots, item lists, or third-party tools like RuneTags.com and BankLayouts.com. RuneLite's layout plugin supports text-based export/import.

What's the difference between official Bank Tags and RuneLite's version?

The official client supports up to 20 tags with basic drag-and-drop organization. RuneLite's version (via the Bank Tag Layouts plugin) adds unlimited tags, one-click presets (Auto-Layout), and export/import functionality. Both are compatible — you can use both simultaneously.

I created a tag but items are missing — what happened?

Make sure you've dragged items onto the tag icon to add them. Simply viewing a tag does not automatically populate it. You need to manually assign each item to the tag.

Can I rename a tag after creating it?

Yes. Right-click the tag icon in the sidebar → select "Rename" → type the new name (up to 25 characters).

Do bank tags work with Potion Storage?

Currently, Bank Tags and Potion Storage are separate systems. To add potions to a tag, you need to withdraw them from Potion Storage into your main bank first, then drag them into the tag. Jagex has noted this as a known limitation.
