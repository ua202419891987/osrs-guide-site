Windrose Loading Screen Stuck & Steam Cloud Save Fix

After a recent patch, some Windrose players get stuck on the loading screen — or the game hangs forever at “Syncing cloud saves” and never reaches the main menu. This guide explains the confirmed causes and gives you recovery steps on PC (Steam) and console.

💾 Save & Launch Fix · 9 min read · Updated July 2026

⏱️ Quick Summary — 30-Second Read

  • 📌 Symptom: infinite loading screen, or stuck at "Syncing cloud saves".
  • 📌 Cause 1: Steam Cloud conflict after a patch changed the save schema.
  • 📌 Cause 2: corrupted local save cache / shader cache.
  • 📌 Cause 3: firewall/antivirus blocking the cloud handshake.
  • 📌 Fix now: verify files, disable Steam Cloud, clear cache, launch offline, then re-enable.

1. What Happens

You hit "Play" in Steam. The game opens, shows the loading bar or the "Syncing cloud saves…" spinner — and then nothing. The bar never completes, the spinner never stops, and you never reach the main menu. Force-closing and re-launching produces the same hang. This is the cloud-sync / save-load deadlock that appears after certain patches change how saves are versioned.

⚠️ Don't Keep Force-Closing

Force-closing mid-sync can leave a half-written save. Do the clean steps below before launching again — it protects your world file.

2. Confirmed Causes

CauseWhy it happensHow common
Steam Cloud schema conflictPatch bumped the save version; Cloud tries to merge old+new and hangsVery common
Corrupted local cacheShader/save cache left from a prior version blocks loadCommon
Firewall / antivirus blockCloud handshake to Steam servers is blocked, so sync never returnsOccasional
Disk write lockAnother process (or the SSD bug) holds the save file openRare

3. How to Fix on PC (Steam)

Run these in order. Most players are back in within 5 minutes.

① Verify Game Files

Steam → Library → Windrose → right-click → Properties → Installed Files → Verify integrity of game files. This replaces any patched file that didn't fully download.

② Disable Steam Cloud (temporarily)

Steam → Library → Windrose → Properties → General → uncheck Keep game saves in the Steam Cloud. This stops the sync handshake that's hanging the launch.

③ Clear the Local Cache

Close Steam. Delete the local cache folders:

  • %LOCALAPPDATA%\Windrose\Saved\ — move (don't delete) the SavedGames subfolder to your desktop first as a backup.
  • Steam → Settings → Downloads → Clear Download Cache.

④ Launch in Offline Mode

Steam → top menu → Go Offline. Launch Windrose. With Cloud off and cache cleared, the game loads the local save directly.

⑤ Re-enable Cloud & Re-sync

Once you're in-game and your world loads, exit cleanly. Go back Online in Steam, re-check the Cloud box, and relaunch — it should now upload the corrected save without hanging.

💡 Reality Check

If it still hangs with Cloud off, the local save itself is corrupt — jump to section 5 (Recovering Your Save) to roll back to the backup you moved to the desktop.

4. How to Fix on Console

Consoles don't use Steam Cloud, but the same "save deadlock" can occur with platform sync.

🎮 Console Recovery Steps

  1. Fully close Windrose (don't just suspend — quit the app).
  2. Disconnect from the network, then launch the game offline so it loads the local save.
  3. Once in, exit cleanly, reconnect, and relaunch — the platform sync re-runs with a valid local save.
  4. If it still hangs, rebuild the console's database / clear the local save cache from storage settings (your cloud backup remains safe).

5. Recovering Your Save

If the active save is corrupt, you have options — in order of safety:

OptionHowRisk
Restore desktop backupCopy the SavedGames folder you moved to desktop back into %LOCALAPPDATA%\Windrose\Saved\None — safest
Steam Cloud rollbackSteam → Windrose → Properties → General → View Steam Cloud → download an older versionLow
New save (last resort)Launch with Cloud off and let the game generate a fresh worldLoses progress
⚠️ Back Up Before You Experiment

Before deleting or overwriting any save, copy the whole Saved folder to an external drive or Desktop. A backup is the difference between a 5-minute fix and a lost world.

6. FAQs

Will disabling Steam Cloud delete my save?

No. Your local save stays on disk. Cloud only controls the online mirror. Re-enabling re-uploads it. Always keep a desktop backup just in case.

Is my world gone forever?

Almost never. The hang is a sync deadlock, not data loss. The local save usually loads fine in offline mode (step 4).

Does this relate to the SSD write bug?

Indirectly. A stuck write lock from heavy disk activity can hold the save file open. If you saw the SSD bug, clear the cache and verify the drive's health first.

Should I reinstall the game?

Only as a last resort. Verify Files (step 1) fixes most patched-file issues without a full reinstall, and reinstalling won't help a Cloud sync deadlock.
