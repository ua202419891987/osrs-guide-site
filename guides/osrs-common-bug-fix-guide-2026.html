OSRS Common Bug Fix Guide 2026: Jagex Launcher, RuneLite & Client Issues

Nothing kills an OSRS session faster than a broken client, black screen, or mysterious crash. This guide covers every common OSRS technical issue and the exact steps to fix them — sourced from Jagex Support documentation, RuneLite Wiki, and community-confirmed solutions.

⚡ Quick Verdict

90% of OSRS technical issues are fixed by one of three things: clear the cache (Jagex Launcher or RuneLite), update your graphics driver, or run the client as administrator. Try these three fixes first before anything else — they save hours of troubleshooting.

1. Jagex Launcher Black Screen

The black screen issue is the most-reported problem with the Jagex Launcher. The launcher opens but shows only a black window. This is caused by cache corruption, display driver conflicts, or hardware acceleration issues. Try these fixes in order:

① Fix 1: Clear Jagex Launcher Cache

Windows Steps

  1. Close the Jagex Launcher completely (check system tray to ensure it's fully closed)
  2. Press Win + R, type %localappdata%\Jagex Launcher and press Enter
  3. Open the cache folder and delete all contents inside
  4. Restart the Jagex Launcher

② Fix 2: Run as Administrator

  1. Right-click the Jagex Launcher shortcut
  2. Select Properties → Compatibility tab
  3. Check "Run this program as an administrator"
  4. Click Apply and OK, then relaunch

③ Fix 3: Reset Internet Settings

The Jagex Launcher relies on embedded browser components. Resetting Internet settings can fix rendering issues:

  1. Open Control Panel → Network and Internet → Internet Options
  2. Go to the Advanced tab
  3. Click Reset under "Reset Internet Explorer settings"
  4. Check "Delete personal settings" and click Reset
  5. Restart the Jagex Launcher

④ Fix 4: Update Your Graphics Driver

Outdated graphics drivers are a common cause of rendering issues in the launcher:

  • NVIDIA: Download GeForce Experience or visit nvidia.com/drivers
  • AMD: Visit amd.com/support for the latest Adrenalin drivers
  • Intel: Visit intel.com/support for integrated graphics drivers
💡 Quick Test: Use the Official Client Instead

If none of the above fixes work, try launching OSRS through the official Old School RuneScape client (not the Jagex Launcher). If the official client works but the launcher doesn't, the issue is specifically with the launcher — submit a support ticket to Jagex with your system specs and a description of the issue.

2. Game Crashing / Crashes on Startup

If the game crashes during loading, shortly after login, or randomly during play, the cause is usually a driver conflict, Java issue, or corrupted cache.

⑤ Fix 1: Update GPU Drivers (Recommended)

This is the most common cause of game crashes. Use a clean driver installation:

  1. Download Display Driver Uninstaller (DDU) from guru3d.com
  2. Boot Windows in Safe Mode
  3. Run DDU to completely remove your current graphics driver
  4. Boot back to normal Windows and install the latest driver from NVIDIA/AMD/Intel

⑥ Fix 2: Disable Third-Party Overlays

Overlays from Discord, Xbox Game Bar, Steam, or NVIDIA GeForce Experience can conflict with OSRS:

  • Discord: Settings → Game Overlay → Disable in-game overlay
  • Xbox Game Bar: Settings → Gaming → Game Bar → Off
  • NVIDIA: GeForce Experience → Settings → General → In-Game Overlay → Off
  • Steam: Settings → In-Game → Disable Steam Overlay

⑦ Fix 3: Increase Virtual Memory

If your system runs low on memory, OSRS can crash without warning:

  1. Right-click This PC → Properties
  2. Click Advanced system settings → Performance Settings
  3. Go to Advanced tab → Virtual memory → Change
  4. Uncheck "Automatically manage paging file size"
  5. Set Initial size and Maximum size to 1.5x your RAM (e.g., 16GB RAM = 24576 MB)
  6. Click Set, OK, and restart your computer

⑧ Fix 4: Clear the Game Cache

For the official client: Press Win + R, type %userprofile%\jagexcache\oldschool\LIVE and delete all contents. Restart the game — it will regenerate the cache fresh.

For RuneLite: Open RuneLite launcher → click the gear icon → Clear cache → Relaunch.

3. RuneLite Not Loading or Crashing

RuneLite is the most popular third-party OSRS client, but it has its own set of common issues.

⑨ Fix 1: Clear RuneLite Configuration

Corrupted plugin settings are a common cause of RuneLite issues:

  1. Close RuneLite completely
  2. Press Win + R, type %userprofile%\.runelite and press Enter
  3. Delete the entire .runelite folder (this resets all plugins and settings)
  4. Launch RuneLite — it will regenerate the folder with default settings

Note: This will reset ALL your plugin configurations, tile markers, and custom settings. If you want to preserve your settings, only delete the cache folder inside .runelite instead.

⑩ Fix 2: Check Java Version

RuneLite requires Java 8 or 11 (64-bit). If you have multiple Java versions installed, conflicts can occur:

  1. Press Win + R, type cmd and press Enter
  2. Type java -version and check the output
  3. If Java is not version 8 or 11 (64-bit), download Java 8 from java.com
  4. Uninstall any other Java versions from Control Panel → Programs and Features

⑪ Fix 3: Run RuneLite as Administrator

  1. Right-click RuneLite.exe (or the shortcut)
  2. Select Properties → Compatibility tab
  3. Check "Run this program as an administrator"
  4. Also check "Run this program in compatibility mode for Windows 8" if the above doesn't work
  5. Apply and relaunch

⑫ Fix 4: Disable Problematic Plugins

If RuneLite launches but crashes when you log in or enter certain areas, a plugin might be the cause:

  1. Launch RuneLite in Safe Mode (hold Shift while launching)
  2. If safe mode works, re-enable plugins one by one until you find the culprit
  3. Common problematic plugins: GPU, HDOS, 117 HD, Animation Smoothing
💡 RuneLite Safe Mode Shortcut

If RuneLite crashes immediately on launch, hold the Shift key while double-clicking the RuneLite icon. This opens the launcher in safe mode with all plugins disabled, letting you fix configuration issues without crashing.

4. Login Issues & Connection Problems

"Still waiting for game response..." and "Login server disconnected" are familiar messages to every OSRS player. Here are the confirmed fixes:

⑬ Fix 1: Switch Worlds

The simplest fix. On the login screen, click Switch World and choose a different server. Total-level worlds (marked with a number like "2,000 Total") often have fewer players and better connectivity.

⑭ Fix 2: Flush DNS Cache

  1. Open Command Prompt as administrator (press Win + X → Command Prompt (Admin) or Terminal (Admin))
  2. Type ipconfig /flushdns and press Enter
  3. Type ipconfig /release then ipconfig /renew
  4. Close the command prompt and restart OSRS

⑮ Fix 3: Change DNS Server

Your ISP's DNS can be slow or unreliable. Switch to a public DNS provider:

  1. Open Control Panel → Network and Sharing Center → Change adapter settings
  2. Right-click your connection → Properties
  3. Select Internet Protocol Version 4 (TCP/IPv4) → Properties
  4. Select "Use the following DNS server addresses"
  5. Enter 8.8.8.8 (Preferred) and 8.8.4.4 (Alternate) for Google DNS
  6. OR enter 1.1.1.1 and 1.0.0.1 for Cloudflare DNS
  7. Click OK and restart your computer

⑯ Fix 4: Disable VPN / Proxy

While a VPN can help with routing issues in some cases, it can also trigger Jagex's anti-cheat systems causing login failures. Temporarily disable any VPN, proxy, or network accelerator while troubleshooting:

  • VPN: Disconnect and try connecting without it
  • Kill Switch / Split Tunneling: Make sure OSRS traffic isn't being routed through a VPN
  • Proxy settings: Windows Settings → Network & Internet → Proxy → Turn off "Use a proxy server"
💡 Check Server Status First

Before doing any complex troubleshooting, check the OSRS server status page (oldschool.runescape.com) or the Jagex Support page (support.runescape.com) to confirm the servers are online. Sometimes the issue is on Jagex's side and no amount of local troubleshooting will help — you just need to wait.

5. High Ping / Lag / Frequent Disconnects

If your game is lagging or disconnecting regularly, the issue is likely with your network connection or the specific game world.

⑰ Fix 1: Use Wired Connection

Wi-Fi is convenient but introduces latency and packet loss. For the most stable OSRS experience, use an Ethernet cable directly to your router.

⑱ Fix 2: Choose the Right World for Your Region

⑲ Fix 3: Add OSRS to Firewall Exceptions

Firewall or antivirus software can block OSRS connections:

  1. Open Windows Security → Firewall & network protection
  2. Click "Allow an app through firewall"
  3. Click "Change settings" then "Allow another app"
  4. Browse to and add: RuneLite.exe, JagexLauncher.exe, and RS2Client.exe
  5. Ensure both Private and Public are checked for each

⑳ Fix 4: Restart Your Router

Sometimes the simplest fix is the most effective. Unplug your router for 30 seconds, plug it back in, wait for it to fully restart (all lights stable), and try again.

6. Graphics Issues: Stuttering, Tearing, Blurry

OSRS's renderer can cause various display issues. Here's how to fix them:

⑴ Fix 1: Enable the GPU Plugin (RuneLite)

RuneLite's GPU plugin dramatically improves performance and visual quality:

  1. Open RuneLite → Plugin sidebar → Search for "GPU"
  2. Enable the GPU plugin
  3. Set Render Distance to 25-50 (higher = more visible area but more GPU load)
  4. Set FPS target to your monitor's refresh rate (60, 144, etc.)
  5. Disable VSync if you experience input lag

⑵ Fix 2: Disable VSync

VSync can cause mouse input lag in OSRS. Disable it in RuneLite's GPU plugin settings. If you want to prevent screen tearing, use Fast Sync (NVIDIA) or Enhanced Sync (AMD) instead through your graphics control panel.

⑶ Fix 3: Adjust Windows Display Settings

  • Fix blurry text: Settings → Display → Scale & layout → Set to 100% (recommended)
  • Fix stuttering: Settings → Gaming → Game Mode → Turn OFF
  • Fix screen tearing: Settings → Display → Advanced display → Choose a lower resolution (e.g., 1280x720) for OSRS

⑷ Fix 4: Check Monitor Connection

If you have a high-refresh-rate monitor (144Hz+), make sure you're using DisplayPort or HDMI 2.0+. Older connections like VGA or DVI can't support higher refresh rates and may cause visual issues.

7. Jagex Launcher Won't Update

When the Jagex Launcher gets stuck on "Checking for updates" or fails to download updates:

⑸ Fix 1: Clean Reinstall

  1. Uninstall the Jagex Launcher from Control Panel → Programs and Features
  2. Delete leftover folders: %localappdata%\Jagex Launcher and C:\Program Files (x86)\Jagex Launcher
  3. Download the latest installer from oldschool.runescape.com
  4. Install and launch

⑹ Fix 2: Check Antivirus

Antivirus programs like Avast, McAfee, or Bitdefender may block the launcher update process. Temporarily disable your antivirus, launch the Jagex Launcher, let it update, then re-enable your antivirus. Or add the entire Jagex Launcher folder to your antivirus whitelist.

⑺ Fix 3: Check Disk Space

The Jagex Launcher needs at least 500 MB of free space on your system drive (usually C:) to download and apply updates. Check your available space in File Explorer → This PC.

8. General Troubleshooting Checklist

When you encounter any OSRS technical issue, run through this checklist first:

🔧 Universal Troubleshooting Order

  1. Restart everything: Close OSRS, close the launcher, restart your computer
  2. Check server status: Visit oldschool.runescape.com to see if servers are online
  3. Clear launcher cache: %localappdata%\Jagex Launcher\cache — delete everything inside
  4. Update graphics driver: Download the latest driver from NVIDIA/AMD/Intel
  5. Run as administrator: Jagex Launcher, RuneLite, and the game client
  6. Disable overlays: Discord, Xbox Game Bar, NVIDIA overlay, Steam overlay
  7. Flush DNS: ipconfig /flushdns in Command Prompt (admin)
  8. Change DNS: 8.8.8.8 / 8.8.4.4 (Google) or 1.1.1.1 / 1.0.0.1 (Cloudflare)
  9. Reinstall: Uninstall and reinstall the client (keep nothing — delete leftover folders)
  10. Submit a ticket: Visit support.runescape.com if nothing works

9. Bug Fix FAQs

Why does my Jagex Launcher show a black screen?

This is caused by cache corruption or display rendering issues. Clear the cache (%localappdata%\Jagex Launcher\cache), run the launcher as administrator, and update your graphics driver. If the issue persists, try resetting Internet settings via Control Panel → Internet Options → Advanced → Reset.

RuneLite keeps crashing when I log in. What's wrong?

A plugin is likely causing the crash. Hold Shift while launching RuneLite to enter Safe Mode (all plugins disabled). If it works, enable plugins one by one until you find the problem. Common culprits include the GPU plugin, 117 HD, HDOS, and Animation Smoothing.

I keep getting "Login server disconnected." How do I fix this?

Try these in order: (1) Switch to a different world, (2) Flush DNS (ipconfig /flushdns in admin CMD), (3) Change your DNS to 8.8.8.8 (Google) or 1.1.1.1 (Cloudflare), (4) Disable any VPN or proxy, (5) Add OSRS to your firewall exceptions.

OSRS crashes without any error message. What do I do?

This is usually caused by old graphics drivers or memory issues. Update your GPU driver using DDU (Display Driver Uninstaller) for a clean install. Then increase your virtual memory to 1.5x your RAM. Disable Discord overlay and Xbox Game Bar as they can cause silent crashes.

My game is laggy even though I have a good PC. How do I fix this?

Enable the GPU plugin in RuneLite if you haven't already. Set render distance to 25-50. Disable VSync in RuneLite settings. Make sure you're using the right world for your region. Close browser tabs and other applications that might be using bandwidth or GPU resources.

Should I use the official client or RuneLite?

RuneLite is Jagex-approved and offers significant quality-of-life improvements (Quest Helper, GPU rendering, XP tracking, tile markers). The official client is more stable but has fewer features. Most experienced players use RuneLite. If you have issues with RuneLite, test with the official client to isolate whether the problem is RuneLite-specific.

I can't log in after upgrading to a Jagex account. What happened?

If you upgraded to a Jagex account, you MUST use the Jagex Launcher to log in. The old official client and direct username/password login no longer work. Download the Jagex Launcher from oldschool.runescape.com, install it, and log in using your new Jagex account credentials.

How do I submit a bug report to Jagex?

Visit support.runescape.com and select the appropriate category for your issue. For technical problems, include: (1) Your system specs (CPU, GPU, RAM, OS version), (2) A detailed description of the issue and when it occurs, (3) Any error messages or screenshots, (4) Steps you've already tried. Submit via the "Contact Us" form at the bottom of the support article.

My RuneLite plugins reset after every update. Can I prevent this?

This usually happens when the .runelite folder is being deleted or corrupted. Back up your settings by copying the entire %userprofile%\.runelite folder to a safe location. Another cause is running RuneLite as administrator — this changes where settings are saved. Run RuneLite normally (not as admin) if possible.

Is there a way to fix OSRS issues without reinstalling?

Yes. Start with the simplest fixes: restart your computer, clear the launcher cache, update your graphics driver, and run the client as administrator. These fix 90% of issues without a reinstall. Only reinstall if the universal troubleshooting checklist (Section 8) fails entirely.

🎯 Quick Troubleshooting Flowchart

Use this decision tree when you hit a technical issue:

❓ Launcher won't open / black screen? → Clear cache → Run as admin → Reset Internet settings → Reinstall
❓ Game crashes on startup? → Update GPU driver → Disable overlays → Increase virtual memory → Safe mode RuneLite
❓ Can't log in / disconnected? → Check server status → Switch worlds → Flush DNS → Change DNS → Disable VPN
❓ Lag / high ping? → Use wired connection → Choose regional world → Firewall exceptions → Restart router
❓ Graphics issues? → Enable GPU plugin (RuneLite) → Disable VSync → Adjust display settings → Check monitor cable

Still stuck? Visit the Jagex Support website (support.runescape.com) or ask for help in the OSRS community Discord server. The community has fixed nearly every issue before — someone has been through exactly what you're experiencing.
