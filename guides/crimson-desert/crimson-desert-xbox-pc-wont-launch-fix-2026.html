Crimson Desert Won't Launch on the Xbox PC App — Fix Guide

You click Play in the Xbox app and… nothing happens. No error, no window, just a spin that quietly gives up. This beginner-friendly guide walks you through the four fixes that resolve the vast majority of "Crimson Desert won't open from the Xbox app" cases — no command line required.

🎮 Beginner Fix · 7 min read · Updated July 2026
Level: Beginner

⏱️ Quick Summary — 30-Second Read

  • 📌 The Xbox app version of Crimson Desert launches through Game Services, not a normal .exe — so a broken cache or stalled service is usually the cause.
  • 📌 Fix #1: Clear the Xbox app cache (Settings → clear local saved data) — fixes ~half of all cases.
  • 📌 Fix #2: Repair / reset Game Services in Windows Settings → Apps.
  • 📌 Fix #3: Run the Xbox app as Administrator so it can launch the game process.
  • 📌 Fix #4: Uninstall and reinstall through the Xbox app only (don't mix with Steam).

1. 🚫 What "Won't Launch" Looks Like

Before we start fixing, make sure you're seeing the Xbox-app flavor of the problem. Players describe it a few ways:

Common symptoms

  • You press Play and the button spins for a second, then returns to "Play" with no window opening.
  • A brief splash appears and instantly disappears (the game process crashes before the menu).
  • Windows shows "Preparing…" forever, or the install button toggles between Install / Play.
  • The Xbox app says the game is installed, but nothing runs — no error code at all.

If you instead get a hard crash with an error code, or a black screen after a logo, a different guide applies. The fixes below target the silent no-launch behavior that is specific to running Crimson Desert through the Xbox app on Windows.

⚠️ One Quick Check First

Confirm your PC meets the minimum spec and that Windows is fully updated. The Xbox app can't launch a game if a required system component (like a Visual C++ runtime or a GPU driver) is missing — but that's rare and usually throws an error. If you see zero reaction at all, continue with the steps below.

2. 🧠 Why the Xbox App Version Behaves Differently

On Steam, Crimson Desert launches as a standard executable. On the Xbox PC app, it launches through Microsoft's Game Services layer — a background service that handles licensing, updates, and process spawning. When that layer gets into a bad state (a corrupted cache, a stuck service, or a permissions hiccup), the game simply refuses to open even though the files are perfectly fine on disk.

The good news: every fix below resets or repairs that Microsoft layer rather than the game itself, so you won't lose save data. Saves for the Xbox app version are stored in your protected Windows profile and survive all four fixes.

💡 Don't Mix Stores

If you also own Crimson Desert on Steam, keep the two installs separate. Running both through the same Windows user can confuse Game Services. Pick one store and stick with it.

3. 🧹 Fix 1 — Clear the Xbox App Cache

This is the single most effective fix and takes under a minute. The Xbox app stores a local cache of game metadata and launch tokens; when it goes stale, the app "forgets" how to start the game.

🪟 Steps (Windows 10 / 11)

  1. Open the Xbox app and click your profile picture (top-left) → Settings.
  2. Select General in the left sidebar.
  3. Scroll to "Clear local saved data / Clear cache" and click it.
  4. The app reloads automatically. Go to My Library → Crimson Desert → Play.

If that alone doesn't help, there's a deeper cache tied to the GamingServices package. Close the Xbox app completely (right-click the icon in the system tray → Quit), then restart your PC and try again. A full restart clears the in-memory service state that the in-app button sometimes misses.

💡 Still nothing?

About 50% of silent-launch cases are resolved by this step. If yours isn't, move on — the next three fixes cover the remaining causes.

4. 🔧 Fix 2 — Repair / Reset Game Services

Game Services is the background process that actually launches the game. If it's corrupted, clearing the cache won't help. You can repair it without touching your games.

🛠️ Repair Game Services

  1. Press Windows key, type Apps & features, and open it.
  2. Scroll to Gaming Services (and Xbox App if listed) → click the three-dot menu → Advanced options.
  3. Click Repair first. This fixes the install without deleting anything.
  4. If Repair doesn't work, click Reset (you'll need to sign back into the Xbox app afterward).
  5. Restart your PC and try launching Crimson Desert again.

Advanced alternative (PowerShell): For stubborn cases, open PowerShell as admin and run:

⌨️ PowerShell command

get-appxpackage Microsoft.GamingServices | remove-appxpackage -allusers

Then reinstall Gaming Services from the Microsoft Store. This is safe and officially supported by Microsoft for launch failures.

⚠️ Admin Note

The PowerShell method requires administrator rights. If you're on a work or school laptop, you may need your IT admin to run it for you.

5. 🛡️ Fix 3 — Run as Administrator

Game Services sometimes can't spawn the game process because it lacks permission to create the window in your session. Forcing the Xbox app to run elevated usually resolves this.

① Set the Xbox app to always run as admin

Right-click the Xbox shortcut → Properties → Compatibility → tick "Run this program as an administrator" → Apply. Do the same for the GamingServices shortcut if you have one.

② Try an elevated launch test

Right-click the Xbox app and choose "Run as administrator" now. Sign in, go to your library, and hit Play. Many "silent no-launch" reports vanish with this step alone.

If running as admin fixes it, keep the compatibility setting on. If it doesn't, don't worry — the last fix reinstalls cleanly and re-registers everything.

💡 UAC Prompt Expected

You'll see a User Account Control prompt each time the app starts elevated. That's normal; click Yes. If you'd rather not see it, the reinstall in the next step often makes it unnecessary.

6. 🔄 Fix 4 — Reinstall via the Xbox App

When cache, service, and permissions are all healthy but the game still won't open, the install itself may be partially corrupted — a failed background update is the usual culprit. A clean reinstall through the Xbox app fixes it.

📥 Clean reinstall steps

  1. In the Xbox app, go to My Library → Crimson Desert → … (more) → Uninstall.
  2. Open Windows Settings → Apps and confirm "Crimson Desert" is fully gone. Reboot.
  3. Back in the Xbox app, find Crimson Desert and click Install. Let it fully download — don't interrupt it.
  4. After install completes, click Play. Sign in if prompted.

Why reinstall through the Xbox app only? The Xbox version uses a different file layout and licensing token than a Steam install. Uninstalling via Windows "Apps & features" alone can leave orphaned tokens that block the next launch. Always use the Xbox app's own uninstall to keep everything consistent.

⚠️ Save Data Is Safe

Xbox app cloud saves sync to your Microsoft account. A reinstall does not delete them. After reinstalling, your progress should re-download automatically when you first launch.

9. ❓ FAQs

Will clearing the cache delete my save?

No. The Xbox app cache only holds metadata and launch tokens. Your Crimson Desert saves are stored separately in your Windows profile and (if enabled) synced to the cloud.

Do I need to reinstall Windows to fix this?

Almost never. The four fixes above resolve the overwhelming majority of cases. A Windows reinstall is overkill and should be a last resort.

Why does it work on Steam but not the Xbox app?

Steam launches the game as a normal executable; the Xbox app routes it through Game Services. A fault in that Microsoft layer affects only the Xbox version. Try the fixes above, or simply play via Steam if you own both.

Resetting Game Services signed me out — is that normal?

Yes. Reset clears the service's stored credentials, so you'll need to sign back into the Xbox app. Your games and saves remain intact.

Should I use a "launch fix" script I found online?

No. Untrusted scripts can modify system services or inject files into your game folder, risking your account and save data. Stick to the official Microsoft/Xbox app steps in this guide.
